HTTP **`421 Misdirected Request`** 客户端错误响应状态码表明，请求被定向到一个无法生成响应的服务器。如果连接被重复使用或选择了其他服务，就有可能出现这种情况。

## 状态

```http
421 Misdirected Request
```
