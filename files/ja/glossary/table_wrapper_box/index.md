**Table Wrapper Box** は [table grid boxes](/ja/docs/Glossary/Table_Grid_Box) の周りに生成されるボックスで、表に表示されるキャプションに必要なスペースを考慮します。絶対位置指定された項目群の包含ブロックが表の場合、このボックスがその項目群の包含ブロックになります。
