**ASCII** (_American Standard Code for Information Interchange_) とは、文字、数字、句読点、および制御コードをデジタル形式に変換するために、コンピュータによって使用される最も一般的な符号化方法の 1 つです。 2007 年から、ウェブ上では {{Glossary("UTF-8")}} が用いられています。
