プログラミング環境において、グローバルスコープとは、他のすべてのスコープを含み、他のすべてのスコープからアクセス可能な{{glossary("scope", "スコープ")}}を指します。

クライアントサイド JavaScript においては、一般的にすべてのコードが実行されている Web ページがグローバルスコープとなります。
