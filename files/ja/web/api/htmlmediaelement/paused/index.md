{{APIRef("HTML DOM")}}

**`HTMLMediaElement.paused`** プロパティは読み取り専用で、メディア要素が一時停止しているかどうかを示します。

## 値

論理値です。 `true` は一時停止中で、`false` は一時停止していません。

## 例

```js
const obj = document.createElement("video");
console.log(obj.paused); // true
```
