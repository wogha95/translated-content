{{APIRef("HTML DOM")}}

**`HTMLMediaElement.muted`** プロパティは、メディア要素がミュートされているかどうかを示します。

## 値

論理値です。 `true` はミュートされていることを意味し、`false` はミュートされていないことを意味します。

## 例

```js
const obj = document.createElement("video");
console.log(obj.muted); // false
```
