{{APIRef("HTML DOM")}}

**`HTMLMediaElement.loop`** プロパティは、 HTML の [`loop`](/ja/docs/Web/HTML/Element/video#loop) 属性を反映し、メディア要素が最後に到達したときに最初からやり直すかどうかを制御します。

## 値

論理値です。

## 例

```js
const obj = document.createElement("video");
obj.loop = true; // true
```
